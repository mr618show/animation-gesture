# animation-gesture

Canvas

This excercise demostrates gesture recognizers and animations

Features:
* Moving Tray with snapping effect
* Panning Faces
* Scaling Faces
* Rotating Faces
* Dropping a Face back on the Tray
* Frictionful Tray
